# Agent P Releases

Release zips for [Agent P](https://github.com/dovannam115/agent-p-releases), the distributed multi-agent workspace.

Inside the app: **Settings → Update → Check now** to install the latest release.

## Manual install

1. Download the latest `AGENT-P_v*.zip` from the [Releases](https://github.com/dovannam115/agent-p-releases/releases) page.
2. Open the app and drop the zip into **Settings → Update → Or install from a local zip**.
3. Click **Restart Agent P**.

Or, the legacy way: copy the zip into the Agent P install folder and double-click `apply-update.bat`.
